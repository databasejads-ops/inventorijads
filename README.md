# Inventori Gudang Sederhana

Isi project:
- index.html — dashboard
- barang-masuk.html — halaman input dan tabel Barang Masuk
- proses-pesanan.html — halaman input dan tabel Proses Pesanan
- stok-realtime.html — stok otomatis per SKU
- styles.css — desain
- app.js — fungsi umum dan data contoh

Cara menjalankan:
1. Ekstrak ZIP.
2. Buka folder.
3. Klik dua kali `index.html`.
4. Gunakan menu di sisi kiri.

Data tersimpan di browser menggunakan localStorage.

Rumus stok realtime:
`Total Barang Masuk - Total Jumlah pada Proses Pesanan`
